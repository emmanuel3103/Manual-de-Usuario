Resumen

El presente documento corresponde al Manual de Usuario del sistema EDI-301, proporcionando las instrucciones básicas de instalación, uso y resolución de problemas comunes. Su objetivo es guiar a los usuarios finales en el manejo eficiente de la aplicación móvil y el sistema web asociados al Programa EDI-301.

Propósito

El propósito de este manual es brindar a los usuarios finales una guía clara, accesible y estructurada que facilite el uso correcto de la aplicación EDI-301. El documento explica cómo instalar la aplicación, cómo navegar dentro de sus módulos, y cómo utilizar las principales funcionalidades disponibles. Asimismo, permite a los usuarios comprender el funcionamiento general del sistema, garantizando una experiencia eficiente y sin complicaciones.

Alcance

Este manual abarca todas las funciones disponibles para los distintos perfiles dentro del sistema EDI-301, incluyendo estudiantes, colaboradores y administradores. Se incluyen instrucciones para la instalación, inicio de sesión, uso de módulos principales como publicaciones, eventos, familias y perfil personal. No aborda temas técnicos de implementación, mantenimiento del sistema o desarrollo de software, los cuales pertenecen al manual técnico y al manual de mantenimiento.

1. Guía de instalación

1.1 Requisitos previos

  • Dispositivo Android 8.0 o iOS 13 o superior.
  • Conexión a Internet estable.
  • Espacio libre mínimo de 100 MB.

1.2 Instalación en dispositivo móvil (Android)

  1. Abrir Google Play Store.
  2. Buscar la aplicación EDI 301.
  3. Presionar Instalar.
  4. Abrir la aplicación una vez descargada.

1.3 Actualización

Para actualizar la app, se puede buscar manualmente una nueva versión en: Configuración → Acerca de EDI 301 → Buscar actualizaciones.

2. Guía de uso de funcionalidades principales

2.1 Inicio de sesión

  1. Abrir la app EDI-301.
  2. Escribir correo institucional y contraseña.
  3. Presionar Iniciar sesión.

2.2 Navegación principal

La barra inferior de navegación incluye los módulos:

  • Inicio
  • Familias
  • Eventos
  • Perfil
  • Notificaciones

2.3 Publicar Evento (solo administradores)

  1. Abrir módulo Eventos.
  2. Presionar + Nuevo evento.
  3. Llenar información requerida.
  4. Presionar Publicar.

2.4 Consultar eventos

  1. Ingresar al módulo Eventos.
  2. Seleccionar un evento para ver su información.
  3. Confirmar asistencia cuando esté disponible.

2.5 Compartir publicación

  1. Acceder al módulo Publicaciones.
  2. Presionar Nueva publicación.
  3. Agregar texto e imagen si se desea.
  4. Publicar.

2.6 Consultar información del perfil

  1. Abrir el módulo Perfil.
  2. Revisar datos personales y actualizables según permisos.

3. Preguntas frecuentes (FAQ)

¿Requiere Internet? Sí, la app necesita conexión activa.

¿Qué hago si olvidé mi contraseña? Usa la opción "¿Olvidaste tu contraseña?".

¿Puedo cambiar mi correo? Solo administrador puede hacerlo.

4. Solución de problemas comunes

La app no inicia sesión: Verifica tu conexión o tus credenciales.

No carga información: Desliza para refrescar o reinicia la app.

No recibo notificaciones: Activa permisos de notificaciones en el dispositivo.

Conclusión

Este manual facilita el uso de la aplicación EDI-301, guiando a los usuarios en instalación, navegación y solución de problemas. Su diseño busca garantizar una experiencia fluida y clara para todos los miembros del programa EDI.
